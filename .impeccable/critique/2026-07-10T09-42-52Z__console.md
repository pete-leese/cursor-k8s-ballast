## Anti-Patterns Verdict

**LLM:** Earned ops familiarity overall, not purple SaaS or neon cyber-ops. Product-slop tells: uppercase micro-labels on every metric/pane title, pill badges, side-stripes on activity/ArgoCD messages, five equal-weight metric cards.

**Deterministic scan:** Directory scan of `console/` returned 0 (`.py` not scannable). Explicit `app.py`/`theme.py` scan: 3× `side-tab` warnings at `console/app.py:575,588,605`. Browser CDP inject: 12 findings (1× `layout-transition` on Streamlit sidebar — FP; 11× `line-length` on `p` — mixed FP/signal). Cursor IDE browser tab could not be held; no Human-tab overlay.

## Overall Impression

IA is right (overview → Investigate problems → RCA). The biggest miss is hierarchy and focus once inside an investigation: Root cause is a fourth tab and a scroll wall, while equal metrics and decorative chrome compete with the failing service.

## What's Working

1. Sidebar Cluster overview vs investigations + Investigate problems as the primary action.
2. Domain-credible status language (Sync/Health, CrashLoop, forward_fix).
3. RCA narrative structure (correlation → resource change → action → evidence) is sound in principle.

## Priority Issues

### [P0] RCA tab is a wall, not a decision
- **Why:** Violates one-job-per-view; demo and triage stall.
- **Fix:** Land on Root cause when RCA exists; collapse expanders; one primary remediation CTA.
- **Suggested command:** `/impeccable distill console` then `/impeccable layout console`

### [P0] Five equal st.metric cards
- **Why:** Hero-metric grid competes with the failing-service signal.
- **Fix:** Hero pod/firing state; demote secondary facts to a dense row.
- **Suggested command:** `/impeccable layout console`

### [P1] Side-stripes + pill monoculture
- **Why:** Absolute ban + detector `side-tab` hits; decorative under stress.
- **Fix:** Drop border-left accents; fewer pills; plain labels / full borders / tints.
- **Suggested command:** `/impeccable quieter console`

### [P1] Tab memory-bridge (Timeline vs Root cause)
- **Why:** Users must recall correlation while deciding action.
- **Fix:** Persistent rollout→alert strip above tabs, or merge highlights into Root cause.
- **Suggested command:** `/impeccable layout console`

### [P2] Projector / contrast hierarchy
- **Why:** Muted `#64748b` uppercase micro-labels fail stage-second readability.
- **Fix:** Darken labels toward ink; bump type; pair color with text for status.
- **Suggested command:** `/impeccable typeset console` / `/impeccable audit console`
